# Ticket: First Jupyter Notebook Setup

## Milestone
M1

## Day
D03

## Goal
Create a notebook that can load and preview the selected dataset.

## Context
Notebook-based exploration helps learners understand data before building scripts.

## Tasks
- Create a first notebook in a clear location (for example `notebooks/`).
- Import required libraries (`pandas` at minimum).
- Load the raw CSV into a DataFrame.
- Display head, shape, and column list.
- Capture first observations in markdown cells.

## Acceptance Criteria
- Notebook runs without errors from start to first preview.
- DataFrame is loaded and basic structure is visible.
- Initial observations are documented.

## Verification
- Run notebook cells top to bottom.
- Confirm head, shape, and columns render correctly.

## OS and Research Note
Jupyter startup commands and environment activation can differ across OS types. If examples fail, research the correct command for your OS and document the command used.

## Notes
Keep notebook focused on inspection, not transformation.
