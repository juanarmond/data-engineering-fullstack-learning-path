# Ticket: Functions and Small Reusable Code

## Milestone
M1

## Day
D03

## Goal
Write one small function in the Week 1 notebook that can be reused instead of copying logic.

## Context
M2 will ask the learner to split work into extract/transform/load functions. This is the beginner version of that habit.

## Tasks
- Open `local/learning-workspace/m1/week-01/notebooks/m1_w1_python_basics.ipynb`.
- Add a `Functions` markdown heading.
- Write one function that takes 1 or 2 inputs and returns a result.
- Call the function at least 3 times with different values.
- Add short comments for the input and returned value if needed.

## Acceptance Criteria
- The learner uses `return`, not only `print`.
- The function is called multiple times with different arguments.

## Verification
- Re-run the function definition cell and the example-call cells.
