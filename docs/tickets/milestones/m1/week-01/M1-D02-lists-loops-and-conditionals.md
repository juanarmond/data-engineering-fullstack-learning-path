# Ticket: Lists, Loops, and Conditionals

## Milestone
M1

## Day
D02

## Goal
Practice repeating logic across a small list of values inside the Week 1 notebook.

## Context
Beginners often understand one value but get stuck when they need to process several rows or names. This day bridges that gap.

## Tasks
- Open `local/learning-workspace/m1/week-01/notebooks/m1_w1_python_basics.ipynb`.
- Add a `Lists/Loops/If` markdown heading.
- Make a list of 5 short strings or numbers related to football matches.
- Loop through the list and print each item.
- Add one `if` statement inside or after the loop.
- Record what confused you and what made sense in `local/learning-workspace/m1/week-01/notes/d02-loops.md`.

## Acceptance Criteria
- The notebook contains one list, one loop, and one conditional.
- The learner can explain what changes each time the loop runs.

## Verification
- Run the new loop cells in the notebook and confirm the outputs appear in order.
