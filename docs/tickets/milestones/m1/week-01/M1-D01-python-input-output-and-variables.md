# Ticket: Python Input, Output, and Variables

## Milestone
M1

## Day
D01

## Goal
Get comfortable working inside a Jupyter notebook and changing simple values on purpose.

## Context
Before SQL and later analysis notebooks, a beginner needs to feel safe editing and rerunning small notebook cells.

## Tasks
- Create `local/learning-workspace/m1/week-01/notebooks/m1_w1_python_basics.ipynb`.
- Add `Setup` and `Variables` markdown headings.
- Add variables for a team name, a match count, and a boolean flag.
- Display those values with short labels in notebook cells.
- Change each value once and rerun the cell.
- Write what changed and what stayed the same in `local/learning-workspace/m1/week-01/notes/d01-python-basics.md`.

## Acceptance Criteria
- The notebook exists and the first cells run successfully.
- The learner can point to three variables and explain what each stores.
- The note explains what output came from the changed values.

## Verification
- Open the notebook and run the `Setup` and `Variables` sections.

## Notes
Keep the notebook tiny. This is about confidence, not complexity.
