# Ticket: Mini Python Data Task

## Milestone
M1

## Day
D06

## Goal
Combine variables, loops, functions, DataFrame work, and a CSV read into one small notebook task.

## Context
This day checks whether the learner can combine the week’s ideas instead of practicing them one by one.

## Tasks
- Continue in `local/learning-workspace/m1/week-01/notebooks/m1_w1_python_basics.ipynb`.
- Add a `Mini Task` markdown heading.
- Read `local/learning-workspace/m1/week-01/data/dummy_matches.csv`.
- Count one simple metric, such as total rows or rows matching a condition.
- Put at least one repeated piece of logic into a function.
- Write what worked and what broke in `local/learning-workspace/m1/week-01/notes/d06-mini-task.md`.

## Acceptance Criteria
- The notebook mini-task cells run end to end.
- The learner can explain which part is input, which part is processing, and which part is output.

## Verification
- Restart the kernel and rerun the notebook through the `Mini Task` section.
