# Milestone M1: Python, SQL, and Reproducible Data Analysis

## High-Level Objective
Build beginner programming and SQL foundations, then use them to understand a real dataset and produce repeatable analysis work.

## What You Will Achieve
By the end of M1, you should be able to:
- use Jupyter notebooks to practice variables, loops, functions, lists, and dictionaries,
- read and write local CSV files with basic Python structure,
- run simple SQL queries inside a beginner-friendly notebook workflow,
- select and scope a dataset for initial analysis,
- organize raw data with clear naming conventions,
- run exploratory analysis in Jupyter,
- compute profile metrics and quality checks,
- document insights and a draft data dictionary,
- make analysis reproducible in a clean notebook flow,
- prepare handoff notes for ingestion automation in M2.

## Weeks In This Milestone
- `week-01/` (Python basics for data work)
- `week-02/` (SQL basics with local data)
- `week-03/` (data foundations and first notebook analysis)
- `week-04/` (notebook reproducibility and M2 handoff)
