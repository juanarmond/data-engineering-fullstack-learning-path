# Milestone M1: Data Foundations and Reproducible Notebook Analysis

## High-Level Objective
Learn to understand a real dataset and build repeatable, notebook-based analysis workflows.

## What You Will Achieve
By the end of M1, you should be able to:
- select and scope a dataset for initial analysis,
- organize raw data with clear naming conventions,
- run exploratory analysis in Jupyter,
- compute profile metrics and quality checks,
- document insights and a draft data dictionary,
- make analysis reproducible in a clean notebook flow,
- prepare handoff notes for ingestion automation in M2.

## Weeks In This Milestone
- `week-01/` (data foundations)
- `week-02/` (notebook-only reproducibility and handoff)
