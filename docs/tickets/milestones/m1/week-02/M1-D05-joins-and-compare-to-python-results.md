# Ticket: Joins and Compare to Python Results

## Milestone
M1

## Day
D05

## Goal
Understand the idea of combining related tables in the notebook and comparing SQL output to earlier Python summaries.

## Context
Even a simple join teaches a key data engineering idea: useful answers often require combining sources.

## Tasks
- Open `local/learning-workspace/m1/week-02/notebooks/m1_w2_sql_basics.ipynb`.
- Add a `JOIN` markdown heading.
- If you have two tables, write one simple join query.
- If you only have one table, create a small lookup table first, then join it.
- Compare one SQL result to a Week 1 Python summary and write the comparison in `local/learning-workspace/m1/week-02/notes/d05-sql-vs-python.md`.

## Acceptance Criteria
- One join query exists in the notebook.
- The note explains whether SQL and Python gave the same answer or why they differed.

## Verification
- Run the join query in the notebook and inspect at least 5 output rows.
