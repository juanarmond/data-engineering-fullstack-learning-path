# Ticket: Quality Queries and Null Checks

## Milestone
M1

## Day
D06

## Goal
Use SQL in the notebook to ask basic data quality questions.

## Context
This prepares the learner for Week 3 profiling and Week 4 reproducibility habits.

## Tasks
- Open `local/learning-workspace/m1/week-02/notebooks/m1_w2_sql_basics.ipynb`.
- Add a `Quality Queries` markdown heading.
- Write queries for at least 3 checks:
  - row count,
  - null count for one important column,
  - duplicate check or missing-value pattern.
- Summarize what looks healthy vs risky in `local/learning-workspace/m1/week-02/notes/d06-quality-queries.md`.

## Acceptance Criteria
- The notebook contains quality-focused queries.
- The note names at least one data risk or one reassuring signal.

## Verification
- Run each query in the notebook and confirm the outputs match the note summary.
