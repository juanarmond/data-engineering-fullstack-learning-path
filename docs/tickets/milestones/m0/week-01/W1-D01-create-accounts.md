# Ticket: Create Core Accounts

## Milestone
M0

## Day
D01

## Goal
Create the accounts required to collaborate in modern software projects.

## Context
Before writing code, a beginner needs identity and collaboration access.

## Tasks
- Create a GitHub account.
- Enable two-factor authentication on GitHub.
- Set profile name and a professional username.

## How To Do It (Basic Steps)
- Go to `https://github.com` and click **Sign up**.
- Confirm your email and complete the account setup.
- Open **Settings -> Password and authentication -> Two-factor authentication**.
- Enable 2FA using an authenticator app.
- Add your display name and a short profile bio.

## Acceptance Criteria
- GitHub account exists and can sign in successfully.
- Two-factor authentication is enabled.

## Verification
- Log in to GitHub in a browser.
- Confirm 2FA status is enabled in account security settings.

## Notes
Use one email dedicated to developer tooling if possible.
