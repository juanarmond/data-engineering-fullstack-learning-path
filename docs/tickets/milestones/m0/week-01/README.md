# Week 01 Summary (M0 Onboarding)

## What You Will Learn This Week

- Create and secure a GitHub account.
- Install and configure Git.
- Use essential CLI terminal commands (`pwd`, `ls`, `cd`, `mkdir`, `git status`).
- Install Python and create a virtual environment.
- Clone a repository and run basic Python commands.
- Create a branch, commit changes, and open a pull request.

## Skills Focus

- Beginner developer setup
- Git/GitHub workflow
- Terminal fundamentals
- Python environment fundamentals
- Basic collaboration habits
