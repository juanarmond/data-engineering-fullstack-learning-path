# Ticket: Install Git and Learn Basic Terminal Commands

## Milestone
M0

## Day
D02

## Goal
Install Git and practice the minimal terminal commands needed for daily work.

## Context
Git and terminal skills are required for almost every engineering role.

## Tasks
- Install Git on your machine.
- Configure `user.name` and `user.email`.
- Practice `pwd`, `ls`, `cd`, and `mkdir`.

## How To Do It (Basic Steps)
- Install Git from `https://git-scm.com/downloads`.
- Open terminal and run `git --version`.
- Set your identity:
  - `git config --global user.name "Your Name"`
  - `git config --global user.email "you@example.com"`
- Practice navigation commands in a test folder:
  - `pwd`, `ls`, `mkdir demo`, `cd demo`, `cd ..`

## Acceptance Criteria
- Git is installed and user identity is configured.
- Learner can navigate directories from terminal without confusion.

## Verification
- `git --version`
- `git config --get user.name`
- `git config --get user.email`

Keep a short personal cheat sheet for terminal commands.
