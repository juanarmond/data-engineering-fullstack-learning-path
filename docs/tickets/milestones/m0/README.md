# Milestone M0: Onboarding Fundamentals

## High-Level Objective
Build the minimum developer setup and collaboration habits needed to start project work safely.

## What You Will Achieve
By the end of M0, you should be able to:
- use GitHub with account security enabled,
- run core terminal commands,
- install and use Python in a virtual environment,
- clone the repository and run basic checks,
- create a branch, commit changes, and open a pull request.

## How To Start
1. Create milestone/week workspace folders:
   - `mkdir -p local/learning-workspace/m0/week-01`
2. Read [week-01/README.md](week-01/README.md).
3. Complete Day 1 to Day 7 in order.
4. Finish the weekly review ticket before moving to M1.

## Exit Criteria
You are ready for M1 when:
- all M0 tickets are complete,
- local setup is stable,
- basic Git workflow is working end-to-end.
