# Code of Conduct

## Our Standards

We expect all contributors to:
- Be respectful and constructive.
- Assume positive intent.
- Give actionable, technical feedback.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Personal attacks in issues, pull requests, or discussions.

## Enforcement

Project maintainers may remove or edit comments, close threads, or block participation for behavior that violates this code.
